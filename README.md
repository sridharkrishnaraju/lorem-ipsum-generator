# Lorem Ipsum Generator

A lorem ipsum generator: make placeholder text by paragraphs, sentences or words, then copy it in one click. Zero dependencies — works in plain HTML, React, Vue, Svelte or Astro.

**▶ [Live demo](https://sgbp.tech/tools/lorem-ipsum-generator)**

```html
<script src="lorem-ipsum.js"></script>
<lorem-ipsum></lorem-ipsum>
```

## What it does

Lorem ipsum is scrambled, Latin-like filler text used to mock up a design before the real copy exists. Because the words carry no meaning, they keep everyone focused on layout, spacing and typography instead of editing the content.

## Install

```bash
npm install @sgbp/lorem-ipsum
```

or copy `lorem-ipsum.js` into your project.
