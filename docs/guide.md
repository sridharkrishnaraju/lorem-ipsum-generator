# Guide: Why designers use placeholder text

Real copy pulls the eye to meaning, which derails a design review. Placeholder text keeps the conversation on hierarchy, line length and rhythm. It also surfaces layout problems early — overflow, awkward wrapping, weak contrast — long before the final words are written.
