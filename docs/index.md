# Lorem Ipsum Generator

A lorem ipsum generator: make placeholder text by paragraphs, sentences or words, then copy it in one click. A free, zero-dependency Web Component you can drop into any site.

```html
<script src="lorem-ipsum.js"></script>
<lorem-ipsum></lorem-ipsum>
```

## What it does

Lorem ipsum is scrambled, Latin-like filler text used to mock up a design before the real copy exists. Because the words carry no meaning, they keep everyone focused on layout, spacing and typography instead of editing the content.

## When you'd use it

Wireframing, building a template, or stress-testing how a layout breathes at different text lengths.

See the [Usage & API guide](usage.md) for framework examples, or the [guide](guide.md) for tips.
