# Usage & API

It is a standard custom element, so it works with no wrapper in plain HTML, React, Vue, Svelte and Astro.

## Plain HTML

```html
<script src="lorem-ipsum.js"></script>
<lorem-ipsum></lorem-ipsum>
```

## React

```jsx
import "@sgbp/lorem-ipsum";
export default function Page() { return <lorem-ipsum />; }
```

## Vue

```vue
<script setup>
import "@sgbp/lorem-ipsum";
</script>

<template>
  <lorem-ipsum />
</template>
```
